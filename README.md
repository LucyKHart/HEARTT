<h1>HEARTT Data Analysis Tool</h1>
<p>This tool was made for the HEARTT Study to analyze the Anonymized Messages created by the HEARTT Data Management Tool. The Analyzer concatenates all sent message content
by month around the unwanted sexual experience, and uses the LIWC to get linguistic markers for that month. It also calculates, number of unique conversations and sent-received ratios</p>

<h2>Installation Instructions</h2>
REQUIREMENTS: These instructions assume you already have either anaconda or anaconda navigator AND Spyder 6 installed. 
For instructions, visit https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html

1. From this github repository download "HeartEnv.yml" 
2. Open Anaconda Navigator, and under the environments tab, select import, then local drive, and then navigate to the downloaded .yml file
   1. If you do not have Anaconda Navigator, but do have conda installed, you can create the environment from the terminal
    <code>conda env create -f [ENVIRONMENT NAME].yml</code>
3. Next, download the HEARTTAnalyzer.py file from this repo
4. Create a folder to stor it in
5. Open Spyder 6 from Applications or Launchpad, not from Anaconda Navigator, if you open it using Anaconda Navigator Spyder will fail to generate the kernel. 
6. Go to the Wrench Icon and set the python interpreter path as the environment path
   1. To get the exact path run <code>conda info --envs</code>
7. You may need to restart the kernel before proceeding
8. Return to the HEARTTAnalyzer.py file and select Open With Spyder 6
9. Use the large green play icon to run the script, and follow the directions in the console
10. If you want to just test if you installed the program successfully, download the AnonymizedDummyData.csv file from this repo. The message content was generated by chatGPT,
I gave chatgpt my anonymized data file, without the 'text' column that held all messages I'd sent, and the contacts all changed to numbers 1-207, so that none of my personal
    information could be accessed by chatGPT, or anyone using this dummy data.
